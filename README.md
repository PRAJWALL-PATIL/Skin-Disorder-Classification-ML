# Skin Disorder Classification using Machine Learning

**Multiclass classification** of erythemato-squamous skin diseases using clinical & histopathological data.

### Dataset
- 366 patient records
- 34 clinical + histopathological features
- 6 skin disease classes

### Models Built & Evaluated
- Logistic Regression, Decision Tree, Random Forest, SVM, KNN

### Results
- **Best model**: Logistic Regression (96% accuracy)
- Delivered practical insights and suggestions for early diagnosis to support dermatologists

### Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook

### Project Workflow
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)
